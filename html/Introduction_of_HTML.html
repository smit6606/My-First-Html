Introduction to HTML

What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and structure webpages. It allows us to define elements like headings, paragraphs, links, and images.

What is DOCTYPE?

The DOCTYPE declaration tells the browser which version of HTML the page is using. It is used to declare an HTML5 document, ensuring that the browser renders the page correctly.

What is html?

The html tag is the root element of an HTML document. It contains all other elements and defines the overall structure of the webpage.

What is head?

The head tag contains metadata about the document, such as its title and links to stylesheets or scripts. This information is not displayed directly on the webpage but is essential for proper rendering and functionality.

What is title?

The title tag defines the document's title, which is shown in a browser's title bar or a page's tab. It helps users identify the page and is also used by search engines to understand the page's content.

What is body?

The body tag represents the content of an HTML document. It contains all the visible elements like text, images, and links.
